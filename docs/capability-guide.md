## 写在前面

经信智能体是由宁波市经济和信息化局开发的专属大模型助手，接入了宁波市工业统计数据仓库、产业知识库、产业链分析平台等内部数据源，能够基于**真实权威数据**回答经信业务相关问题。

**一个重要特点**：本智能体严格遵循"有数据才说话"的原则。如果查询不到相关数据，会如实告知，而不会凭空编造内容。所有关键数据都会标注来源，确保回答可核查、有依据。

## 一、工业经济数据查询

### 能做什么

从宁波市工业统计数据仓库中查询精确的经济运行指标数值，包括但不限于规模以上工业增加值、工业总产值、利润总额、营业收入、从业人员、出口交货值等，支持按行业、时间维度筛选。

### 适用场景

- 撰写经济运行分析报告时需要具体数据支撑
- 汇报材料中需要核实某项指标的准确数值
- 月度、季度、年度经济运行情况跟踪
- 重点行业数据核查与对比

### 常见提问示例

**基础数据查询**
```
2024年宁波市规上工业增加值是多少？
2025年1-2月宁波市规上工业总产值是多少？
2024年宁波工业利润总额同比增长多少？
今年一季度宁波规上工业增速是多少？
```

**行业分类查询**
```
2024年宁波市汽车制造业产值是多少？
去年宁波计算机、通信和其他电子设备制造业营业收入情况如何？
宁波纺织业近三年从业人员变化趋势如何？
2024年宁波战略性新兴产业增加值占规上工业比重是多少？
```

**时间对比查询**
```
2022年和2023年宁波规上工业增加值分别是多少？
近五年宁波工业增速走势怎么样？
2024年与2020年相比，宁波工业总产值增长了多少？
```

**排名查询**
```
宁波主要工业行业中哪个行业产值最高？
2024年宁波各区县规上工业增加值排名如何？
```

### 提问技巧

- 指定**具体年份或月份**，如"2024年""2025年1-3月"
- 指定**具体行业名称**，使用官方统计分类，如"规模以上工业""战略性新兴产业""高技术制造业"
- 一次问一个指标效果最好；如果同时问多个指标，智能体会自动拆分分别查询

## 二、产业链全景分析

### 能做什么

调用宁波产业知识中心的产业链分析平台，获取重点产业链的全景画像，包括产业规模、企业分布、核心企业、技术创新、投融资动态、上下游链条结构等综合分析内容。

### 当前支持的产业链

| 产业链名称 | 可查询内容 |
|-----------|-----------|
| **新能源汽车** | 整车/电池/电机/电控企业分布、核心企业画像、融资动态、技术专利 |
| **新一代人工智能** | AI企业总量、上市公司、技术领域、融资规模与趋势 |
| **人形机器人** | 企业分布、核心技术、资本动向、上下游链条 |
| **机器人（通用）** | 工业机器人、协作机器人企业、专利、融资 |
| **先进石油化工** | 化工新材料、精细化工企业画像、技术路线 |
| **智能家电** | 品牌企业、供应链、智能化转型动态 |
| **智能座舱** | 舱内屏幕、HMI、域控制器等企业与技术 |
| **智慧物流** | 无人配送、自动化仓储、供应链科技企业 |

### 可获取的具体内容

**产业规模指标**
- 产业链节点数量、企业总数
- 上市企业数量（含科创板/创业板/北交所）
- 近五年专利授权总量
- 近五年投融资总金额

**企业画像**
- 重点企业清单（龙头/专精特新/独角兽）
- 注册资本规模分布
- 区域/城区分布情况
- 产业链环节分布（上游/中游/下游）

**技术创新动态**
- 近五年专利授权数量走势
- 技术生命周期分析
- 核心技术领域 TOP 8

**投融资态势**
- 近五年融资规模走势
- 热门投资城市
- 资本关注重点方向

**产业链图谱**
- 上下游链条结构（树状展示）

### 常见提问示例

**产业总体情况**
```
介绍一下宁波新能源汽车产业的总体发展情况
宁波AI产业目前有多少企业？有哪些上市公司？
宁波人形机器人产业的发展现状怎么样？
宁波智能家电产业在全国处于什么水平？
```

**企业分布情况**
```
宁波新能源汽车产业的核心企业有哪些？
宁波机器人企业主要集中在哪些区县？
宁波AI产业的专精特新企业有哪些？
新能源汽车产业链上下游各有哪些宁波企业？
```

**技术与创新**
```
宁波新能源汽车产业近五年专利授权情况如何？
宁波AI产业的核心技术领域集中在哪些方向？
宁波人形机器人产业的技术路线是什么？
```

**投融资情况**
```
近年来宁波AI产业融资规模有多少？
宁波机器人产业投融资主要集中在哪些细分方向？
新能源汽车产业近几年融资趋势是上升还是下降？
```

## 三、产业资讯与动态

### 能做什么

获取各重点产业链的最新动态资讯，涵盖政策发布、企业融资、技术突破、产品发布、产业活动等多种类型，支持按产业链、地区、资讯类型筛选。另外有专门的 AI 热点聚合功能，可快速获取本周 AI 领域的重要事件。

### 资讯覆盖类型

| 资讯类型 | 说明 |
|---------|------|
| **政策动向** | 国家/省/市级产业扶持政策发布 |
| **融资报道** | 重点企业融资轮次与金额 |
| **技术突破** | 关键技术攻关成果、科研发布 |
| **产品发布** | 新产品、新解决方案上市 |
| **产业活动** | 行业峰会、展会、论坛动态 |
| **头部企业动态** | 龙头企业战略、扩产、合作消息 |

### 常见提问示例

**产业动态**
```
最近新能源汽车产业有哪些重要动态？
近期国内机器人行业有什么重大政策出台？
宁波AI产业最新的融资情况有哪些？
最近人形机器人领域有什么技术突破新闻？
```

**按类型筛选**
```
最近有哪些关于智能制造的政策出台？
近期新能源汽车有哪些重要融资事件？
宁波智慧物流行业最近有哪些大型展会？
```

**按地区筛选**
```
宁波本地新能源汽车企业最近有什么动态？
全国范围内AI产业最近的重要事件有哪些？
省内机器人产业最近的政策有哪些？
```

**AI 热点聚合**
```
本周AI领域有哪些热点事件？
最近一周AI产业有什么大新闻？
帮我汇总一下近期AI行业的重要动态
```

## 四、政策文件检索

### 能做什么

从知识库中语义检索政府政策文件、法规条文、扶持办法、实施细则等，支持模糊提问，能够理解政策意图并定位相关内容，而无需记住文件的准确名称。

### 可检索的内容类型

- 国家工信部、省工信厅、宁波市经信局发布的产业政策
- 各类专项扶持资金申报办法与细则
- 企业认定标准（专精特新、高新技术企业等）
- 数字化转型、智能化改造相关指导意见
- 绿色制造、节能减排政策文件

### 常见提问示例

**政策查询**
```
宁波市支持工业企业数字化转型有哪些政策？
国家关于专精特新企业的认定标准是什么？
宁波市制造业高质量发展有哪些扶持办法？
关于工业互联网平台建设的政策有哪些？
```

**补贴与申报**
```
宁波工业技改资金补贴的申请条件是什么？
智能化改造有哪些可以申请的资金支持？
新能源汽车产业有哪些专项扶持政策？
工业互联网标杆工厂怎么申报？
```

**政策解读**
```
"专精特新"和"小巨人"企业的区别是什么？
宁波市支持人工智能产业发展的政策主要涵盖哪些方面？
工信部关于数字经济的最新政策导向是什么？
宁波市碳达峰实施方案对工业领域有哪些要求？
```

**文件原文**
```
帮我找一下宁波市关于制造业数字化转型的实施方案
宁波市工业企业技术改造资金管理办法的主要内容是什么？
关于加快推进宁波市工业互联网发展的相关文件有哪些？
```

## 五、产业报告查阅

### 能做什么

从知识库中检索和查阅各类产业分析报告、行业研究报告、年度发展报告等非结构化文档，摘取关键内容并整合成清晰的回答。

### 可查阅的报告类型

- 宁波市各产业链年度发展报告
- 各行业运行分析报告
- 企业发展情况调研报告
- 重点园区、产业基地发展报告
- 产业政策研究报告

### 常见提问示例

**查阅报告内容**
```
宁波新能源汽车产业的最新年度报告说了什么？
宁波软件和信息技术服务业发展报告的主要结论是什么？
最近有没有关于宁波智能制造发展的研究报告？
2024年宁波工业经济运行分析报告的核心数据是什么？
```

**提取关键信息**
```
帮我找一下关于宁波汽车产业"十四五"规划的内容
宁波制造业转型升级的主要挑战有哪些？（从报告中总结）
知识库中有没有关于宁波数字经济发展的报告？
```

> **注意**：如果知识库中没有对应的报告，智能体会如实告知，不会凭空生成报告内容。

## 六、数据计算与分析

### 能做什么

在成功获取工具数据的前提下，对数据进行同比/环比计算、增速分析、贡献率计算、占比计算、跨年趋势分析等，并给出清晰的计算过程和结论。

### 支持的计算类型

| 计算类型 | 说明 | 示例问法 |
|---------|------|---------|
| **同比增速** | 与上年同期相比的增长率 | "2024年规上工业增速如何？" |
| **环比增速** | 与上月/上季度相比的增长率 | "本季度与上季度相比增速变化" |
| **占比/比重** | 某指标占总体的百分比 | "高技术制造业占规上工业比重" |
| **贡献率** | 某行业对整体增长的贡献度 | "哪个行业对今年工业增长贡献最大？" |
| **拉动点数** | 某行业拉动整体增长的百分点数 | "汽车制造业拉动工业增速几个百分点？" |
| **趋势分析** | 多年数据的走势分析 | "近五年宁波工业增速变化趋势" |
| **排名对比** | 多个行业/地区数据对比排序 | "各行业产值排名" |
| **单位换算** | 千元→万元→亿元自动换算 | 自动处理，无需手动指定 |

### 常见提问示例

**增速计算**
```
2024年宁波规上工业增加值同比增速是多少？
今年1-6月工业增速与去年同期相比有多大变化？
宁波汽车制造业产值近三年年均增速是多少？
```

**占比与贡献率**
```
2024年高技术制造业增加值占宁波规上工业比重是多少？
战略性新兴产业对宁波工业增长的贡献率有多高？
汽车制造业拉动宁波工业增速约多少个百分点？
```

**多维对比分析**
```
对比2022、2023、2024年宁波工业利润总额的变化
宁波与杭州、温州规上工业增速对比如何？
近五年宁波战略性新兴产业占比变化趋势分析
```

**综合分析**
```
帮我分析一下2024年宁波工业经济的总体运行情况
今年上半年宁波工业经济有什么亮点和不足？
从数据来看，宁波工业结构调整进展如何？
```

## 七、图表与可视化

### 能做什么

在成功获取真实数据后，调用图表生成工具生成折线图、柱状图、饼图等多种类型的可视化图表，图表自动在页面中展示，并可下载保存。

> **重要**：图表必须基于真实查询数据生成，不接受无数据的"虚构图表"请求。

### 支持的图表类型

| 图表类型 | 适用场景 |
|---------|---------|
| **折线图** | 展示时间序列趋势，如多年增速变化 |
| **柱状图** | 比较不同行业/地区/时期的数据大小 |
| **分组柱状图** | 同时比较多个指标的多组数据 |
| **饼图** | 展示各部分占总体的比重结构 |
| **堆叠柱状图** | 展示整体构成的变化 |
| **面积图** | 展示数量随时间的累积变化 |

### 常见提问示例

**趋势图**
```
画一个折线图，展示2020-2024年宁波规上工业增加值变化趋势
帮我生成近五年宁波工业增速走势图
画出2019-2024年宁波高技术制造业产值变化的折线图
```

**对比图**
```
帮我画一个柱状图，对比2024年宁波各主要工业行业的产值
生成一个图表，比较近三年宁波与全省工业增速
画出汽车、电子、纺织三个行业2022-2024年产值对比图
```

**结构图**
```
用饼图展示2024年宁波规上工业各行业产值占比
生成一个图，展示宁波战略性新兴产业与传统产业的比重结构
帮我画宁波工业利润的行业构成饼图
```

**综合分析图**
```
帮我生成一份宁波工业经济运行情况的综合可视化分析
用图表展示宁波AI产业的企业数量和融资金额变化
```

### 提问技巧

- 可以先问数据，确认数据准确后再要求画图
- 指定图表类型（折线/柱状/饼图）会让结果更精准
- 如果自己手上已有数据，可以直接提供数据让智能体画图：
  ```
  请用以下数据画折线图：
  2020年：1200亿，2021年：1350亿，2022年：1420亿，2023年：1580亿，2024年：1720亿
  ```

## 八、报告撰写与导出

协助撰写产业分析报告、经济运行分析报告、汇报材料等，基于真实数据自动整合成规范的文档结构，并支持导出为 Word（.docx）或 Excel（.xlsx）格式，可直接下载使用。

### 报告导出格式

| 格式 | 适用场景 | 特点 |
|------|---------|------|
| **Word 文档** | 正式分析报告、汇报材料 | 方正字体排版，可直接修改打印 |
| **Excel 表格** | 数据汇总表、统计明细 | 自动格式化，表头蓝色底色，交替行色 |

### 常见提问示例

**报告撰写**
```
帮我写一份2024年宁波工业经济运行情况分析报告
帮我生成一份宁波新能源汽车产业发展报告（含现状、问题、建议）
帮我撰写一份关于宁波AI产业发展的调研报告提纲
帮我起草一份2025年一季度工业经济形势分析材料
```

**导出 Word**
```
把这份分析内容导出为Word文档，我需要打印
帮我生成一份宁波制造业数字化转型分析报告，并导出为Word
```

**导出 Excel**
```
把这个数据表格导出为Excel
帮我把宁波各区县工业增加值数据整理成Excel表格
将以下统计数据导出为Excel文件
```

**辅助撰写**
```
帮我补充一下这份报告中的数据部分
给这个汇报材料加上相关数据支撑
帮我把这段描述改得更专业一些
```

## 九、文件附件分析

### 能做什么

支持上传文件（Word、PDF、Excel、图片等），智能体读取文件内容后，可以对文件进行摘要提取、内容分析、数据整理、问题解答等操作。

### 适用场景

- 上传一份报告，让智能体提炼核心观点
- 上传一份数据表格，让智能体分析数据规律
- 上传政策文件，让智能体帮你理解关键条款
- 上传会议纪要，让智能体提取行动事项

### 常见使用方式

```
[上传文件] 帮我总结一下这份报告的主要内容
[上传文件] 这份政策文件中，关于补贴申请的条件是什么？
[上传文件] 帮我分析这张数据表中有哪些异常值
[上传文件] 这份会议纪要里有哪些需要经信局跟进的事项？
[上传文件] 对比这两份报告，主要差异在哪里？
```

### 支持的文件格式

- 文档类：`.pdf`、`.docx`、`.doc`、`.txt`
- 表格类：`.xlsx`、`.xls`、`.csv`
- 图片类：`.png`、`.jpg`、`.jpeg`

## 十、互联网信息补充

### 能做什么

当内部数据库和知识库都无法满足需求时，自动调用互联网搜索补充公开信息，用于获取国内外产业动态、公开统计数据、行业报告等。

> **注意**：互联网搜索结果仅作为内部数据的补充，可靠性低于数仓和知识库，重要数据以内部权威数据为准。

### 常见使用场景

```
最新的全国规上工业增速是多少？（内部数据仅有宁波数据时）
国家最新出台了哪些关于工业互联网的政策？
国际上人工智能产业最新的发展动态有哪些？
最近工信部有没有关于数字化转型的新政策？
```

## 十一、使用提示与注意事项

### 提问技巧

**1. 指定时间范围**
- 尽量说清楚时间，例如："2024年""2025年1-3月""近五年"
- 避免模糊的"最近""近期"，如果确实不知道具体年份，可以问"最新数据是哪年的？"

**2. 指定行业口径**
- 使用统计部门的规范名称，如"规模以上工业""战略性新兴产业""高技术制造业"
- 避免口语化表述，如"大的工厂""高科技企业"

**3. 分步提问效果更好**
- 先查基础数据 → 再要求分析 → 最后要求画图或导出
- 复杂问题可以拆解成几个小问题逐步推进

**4. 明确你想要什么**
- 要数字：直接问"XXX是多少？"
- 要分析：问"请分析一下XXX的情况"
- 要报告：问"帮我写一份XXX报告"
- 要图表：问"帮我画一个XXX的图表"
- 要导出：说"导出为Word/Excel"

### 关于数据准确性

- 所有回答中的关键数据都标注了来源（数据库表名/文档名），你可以用来核实
- 如果智能体说"未查询到数据"，是真的没有，不是在推脱
- 遇到数据可疑时，可以直接问："这个数据是从哪个数据库查出来的？"

### 关于能力边界

**智能体擅长的**：
- 宁波市工业经济相关数据的精确查询
- 上述支持产业链的全景分析
- 经信局日常工作中的数据支撑与报告撰写

**智能体不擅长的**：
- 数仓中没有录入的数据（会如实告知）
- 不在支持列表中的产业链深度分析
- 2024年底以后的最新政策（知识库更新有滞后）
- 个人隐私、企业内部保密信息

## 十二、常见问题 Q&A

**Q：问了半天智能体说"没有数据"，是不是系统有问题？**

A：不是系统问题。数仓和知识库的数据有其覆盖范围，如果你问的指标或行业不在收录范围内，系统会如实告知。可以换个角度提问，或询问"目前系统里有哪些关于XX行业的数据？"

---

**Q：智能体回答的数据和我手里的报告数据不一致，以哪个为准？**

A：注意查看智能体回答中标注的"来源"（数据库表名/文档名），确认是否与你的报告来源相同。数据口径不同（如统计时间节点、行业分类标准）可能导致数值差异。如有疑问，建议联系数据维护部门确认。

---

**Q：可以上传文件让智能体帮我分析吗？**

A：可以。在聊天输入框的附件按钮处上传文件，然后告诉智能体你想要什么分析。目前支持 PDF、Word、Excel、图片等常见格式。

---

**Q：可以让智能体帮我写汇报材料吗？**

A：可以，但效果取决于相关数据是否在数仓或知识库中。建议先查询相关数据，确认有数据支撑后，再让智能体帮你起草报告。生成的文档可以导出为 Word 格式进一步修改。

---

**Q：问产业政策时，智能体的回答是否是最新的？**

A：知识库中的政策文件有更新周期，最新发布的政策可能还未收录。建议通过"互联网搜索"补充查询最新动态，但要留意互联网信息的可靠性需自行判断。

---

**Q：怎么让智能体生成图表？**

A：直接说"帮我画一个XXX的图表"即可。如果需要特定类型，可以指定："画折线图""画柱状图""画饼图"。智能体会先查数据，再生成图表，图表会直接在对话中展示，也可以下载保存。

---

**Q：生成的 Word 报告可以直接使用吗？**

A：生成的 Word 文档采用方正字体标准排版，可以直接用于工作汇报。但建议在使用前仔细核对数据，并根据实际需要调整格式和措辞。智能体生成的是初稿，最终内容由使用者负责。

---

**Q：能帮我做竞争城市（如杭州、苏州）的对比分析吗？**

A：这取决于数仓是否有其他城市的数据。如果数仓只有宁波数据，则无法对比；这时候智能体可以通过互联网搜索补充其他城市的公开数据，但数据的权威性和精度不如内部数仓。

---

**Q：可以一次问多个问题吗？**

A：可以。例如："帮我查一下2024年宁波规上工业增加值、利润总额和营业收入"，智能体会自动拆分成多次工具调用后整合回答。但复杂的综合性问题建议分步提问，每步确认后再推进，效果会更好。
