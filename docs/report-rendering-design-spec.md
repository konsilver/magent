# 报告渲染 UI 设计规范

---

## 报告正文排版规范

### 4. h1 标题
- **示例**：「人形机器人产业链深度全景分析报告」
- **字体样式**：加粗（Bold），#262626
- **字号**：20px
- **行高**：28px
- **间距**：距离上方8px、下方10px
- **对齐**：左对齐

### 5. h2 标题
- **示例**：「一、产业链概述」「二、产业链结构图谱」「三、企业分布分析」
- **字体样式**：加粗（Bold），#262626
- **字号**：18px
- **行高**：25px
- **间距**：上下间距8px
- **对齐**：左对齐

### 6. h3 标题
- **示例**：「上游（81.3%，785家企业）」「中游（16.5%，159家企业）」
- **字体样式**：加粗（Bold），#262626
- **字号**：16px
- **行高**：22px
- **间距**：上间距8px，下间距6px
- **对齐**：左对齐

### 7. 段落（正文）
- **字体样式**：常规（Regular），#262626
- **字号**：16px
- **行高**：24px
- **段间距**：段落上下间距8px

## 三、结构化内容元素

### 8. 表格
- **表头**：
  - 列名示例：「链接」「文件大小」「时间」
  - 表头背景色：#E9ECF0
  - 表头文字：#808080，加粗（Bold）
  - 表头行高：标题上下加14px间距
- **表格行**：
  - 白色背景，行间有（`1px solid #E9ECF0`）描边分割
  - 文字样式：正文字号（16x），常规字重
  - 表头行高：标题上下加14px间距
- **表格边框**：外边框有细线（`1px solid #E3E6EA`）整体圆角12px
- **列宽**：根据内容进行均匀排布
- **表格间距**：上间距8px，下间距12px

### 9. 无序列表
- **样式**：黑色实心圆点（`•`）作为列表标记
- **缩进**：不进行缩进
- **示例**：
  - • 大脑和小脑：芯片、控制器、算法、操作系统、应用程序
  - • 执行系统：电机、减速器、丝杠、一体化关节
  - • 感知系统：视觉传感器、触觉传感器、姿态传感器、力矩传感器、听觉传感器
- **文字样式**：与正文一致（16px，常规字重，行高24px）
- **行间距**：列表项之间间距4px
- **间距**：整个列表和上下文之前上间距2px，下间距8px

### 10. 有序列表
- **样式**：阿拉伯数字 + 英文句号（`1.` `2.` `3.`）作为列表标记
- **缩进**：不进行缩进
- **示例**：
  1. 大脑和小脑：芯片、控制器、算法、操作系统、应用程序
  2. 执行系统：电机、减速器、丝杠、一体化关节
  3. 感知系统：视觉传感器、触觉传感器、姿态传感器、力矩传感器、听觉传感器
- **文字样式**：与正文一致（16px，常规字重，行高24px）
- **行间距**：列表项之间间距4px
- **间距**：整个列表和上下文之前上间距2px，下间距8px

### 11. 引用源（Citation）
- **位置**：出现在有引用源标记的地方，行内角标形式
- **样式**：灰色小标签/角标，圆角100%，背景色#3F5F6F7，示例为「注1」
- **字号**：小于正文（14px）
- **行高**：22px
- **颜色**：灰色#1808080
- **交互**：可点击，弹出/跳转到引用来源

### 12. 分析模块（高亮区块）
- **背景色**：浅蓝色（ `#EBF2FF`）
- **圆角**：12px
- **内边距**：上下12px，左右16px
- **描边**：左侧3px，#126DFF，实线描边
- **文字内容**：以「分析：」开头，后接分析总结文本
- **文字样式**：正文字号，常规字重，黑色

### 13. 横线（分割线）
- **线条样式**：1px，实线线条，#E3E6EA
- **间距**：上下间距30px

### 14. 引用
- **字体样式**：常规（Regular），#262626
- **字号**：16px
- **行高**：24px
- **内间距**：左侧16px
- **描边**：左侧2px，实线，#E3E6EA
- **模块间距**：上下16px

### 15. 图片
- **比例**：按图片原始比例缩放
- **间距**：上下16px，左12px，右16px

## 四、整体页面布局

### 左侧边栏
- 宽度约 280px
- 包含：新建对话按钮、智能体列表、历史对话列表
- 背景色：#F7F7F7


### 内容区
- 内容区最大宽度约 840px，水平居中
- 左右 padding 约 24–40px

| 代码块/引用块背景 | `#F5F6F7` |
| 分析模块高亮背景 | `#EBF2FF` |
| 表头背景 | `#F5F6F7` |

## 六、字体层级

| 层级 | 字号（近似） | 字重 | 用途 |
|---|---|---|---|
| h1 | 20px | Bold | 报告主标题 |
| h2 | 18px | Bold | 一级章节标题（一、二、三…） |
| h3 | 16px | Bold | 二级章节标题（上游、中游…） |
| 正文 | 16px | Regular | 段落、列表项 |
| 引用源角标 | 14px | Regular | 蓝色角标「注1」 |
| 代码块 | 14px | Regular (Monospace) | JSON/代码展示 |
| 表格内容 | 16px | Regular | 表格单元格 |
