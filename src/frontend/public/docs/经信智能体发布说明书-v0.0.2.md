## 这是什么

经信智能体是面向经信业务场景的对话助手。它的核心目标不是“聊天”，而是帮助你更快完成以下工作：

- 查找和核对工业经济数据。
- 检索政策、报告和知识库材料。
- 做结构化分析并生成图表。
- 输出可下载的报告材料（Word、Excel）。

它遵循一个基本原则：优先基于工具检索到的真实数据回答，查不到就明确告知，不编造。

## 模型所拥有的能力

从使用体验上看，你主要会感知到两种模式：

- 快速模式：响应更快，适合日常问答、快速查数、简要说明。
- 思考模式：更适合复杂任务，适合多步骤分析、对比、总结、报告撰写。

系统支持按配置接入不同底层模型（当前前端可选 Qwen / DeepSeek），但你不需要学习模型参数，重点只要把业务问题说清楚。

## 系统内置工具（后台自动调用）

你在前台看到的是一个对话框，后台会自动调用工具完成任务。当前主要工具如下：

| 工具名称 | 用途（业务理解） |
| --- | --- |
| 数据库查询 | 查询工业经济指标、统计数值等“硬数据” |
| 知识库检索 | 检索政策文件、报告、制度材料 |
| 互联网搜索 | 当内部没有数据时，补充公开信息 |
| 产业链信息 | 获取产业链画像、产业动态、AI 热点 |
| 数据可视化 | 把数据自动转成图表 |
| 报告导出 | 将内容导出为 Word 或 Excel |

## 系统内置技能（回答更稳的工作方法）

这些技能是系统在后台使用的“工作规范”，帮助输出更稳定、可复核：

| 技能名称 | 作用 |
| --- | --- |
| 能力清单速答技能 | 回答“这个系统能做什么、怎么问” |
| 快速学习分析材料 | 对政策、规划、报告等材料做结构化提炼 |

其余业务技能会在**能力中心**中动态展示，包括事项办理、政策解读、
企业画像、产业链分析、材料对比和指标查询等能力。

## 示例提问（可直接复制）

### 数据查询类

- `2024年宁波规上工业增加值是多少？`
- `2025年1-3月规上工业总产值同比是多少？`
- `近五年宁波工业增速走势如何？`

### 产业链分析类

- `请分析宁波新能源汽车产业当前发展情况。`
- `宁波机器人产业近五年融资趋势如何？`
- `宁波AI产业有哪些代表性企业？`

### 政策与知识库类

- `宁波支持工业企业数字化转型有哪些政策？`
- `请帮我梳理工业技改资金申报条件。`
- `从知识库里总结“智能制造”相关报告的核心结论。`

### 计算与可视化类

- `帮我计算2024年高技术制造业占规上工业比重。`
- `把2020-2024年规上工业增加值做成折线图。`
- `把刚才的数据做成柱状图并导出。`

### 报告与文件类

- `基于上面的分析，生成一份简版汇报并导出Word。`
- `把表格整理成Excel导出。`
- `我上传了两份材料，请对比差异并列出重点变更。`

## 能做什么与不能做什么（能力边界）

### 能做什么

- 能基于已接入的数据源和知识库给出有依据的回答。
- 能做查询、分析、图表和材料输出。
- 能在信息不足时明确提示，不强行下结论。

### 不能做什么

- 不能替代正式审批、行政决定、法律裁量。
- 不能保证所有互联网信息都绝对准确或最新。
- 不能在没有数据依据的情况下给出精确数字结论。
- 不能完整解析所有格式附件（当前重点支持 PDF、DOC、DOCX、WPS、TXT 文本解析；单文件上传上限 50MB）。

## 推荐使用方式

建议你按下面四步提问，结果会更准确：

1. 先说清时间范围：例如“2024年”或“2025年1-3月”。
2. 再说清口径对象：例如“规上工业”“高技术制造业”。
3. 明确任务目标：查数值、做分析、画图、导出报告。
4. 复杂任务分步提：先查数据，再分析，再出图或导出。

## 常见问题

### 为什么有时会提示“暂无数据”？

这通常表示当前接入的数据源里没有可用结果。系统会优先保证准确性，而不是猜测结果。

### 我想要更快回复怎么办？

可以优先使用快速模式，适合日常查询和简要说明。

### 我想要更完整分析怎么办？

使用思考模式，并把背景、时间范围、目标写完整，系统更容易给出结构化结果。
