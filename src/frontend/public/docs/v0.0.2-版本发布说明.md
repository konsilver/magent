# 经信智能体 v0.0.2 版本维护公告

尊敬的各位同事：

经信智能体已于 **2026年2月27日** 完成 `v0.0.2` 版本升级。  
本次升级重点围绕“更好用、更稳定、结果更可核查”开展优化。

## 一、这次升级带来了什么变化

### 1. 能力中心更清晰，找功能更方便

- 能力中心展示更完整，技能、工具、知识库分类更明确。
- 查看可用能力时，更容易判断“该问什么问题、用什么能力”。

### 2. 知识库使用体验提升

- 知识库列表和文档列表展示更清楚。
- 支持查看文档详情，便于快速确认材料内容。
- 政策、报告类问题在知识库检索场景下更稳定。

### 3. 新增“文件上传 + 文件解析 + 对话分析”闭环能力

- 可上传常见材料并结合提问进行分析。
- 当前重点支持：`PDF`、`DOC`、`DOCX`、`WPS`、`TXT`。
- 单个文件上传上限为 **50MB**。

### 4. 对话模式可切换，兼顾速度和深度

- 新增“快速模式”：适合日常问答、快速查数。
- 新增“思考模式”：适合复杂分析、报告整理、材料对比。
- 可按任务场景选择模式，提高工作效率。

### 5. 输出能力继续增强

- 数据分析结果可继续生成图表并下载。
- 报告可导出为 `Word`，表格可导出为 `Excel`，便于直接用于材料整理和汇报。

### 6. 系统稳定性与运维能力优化

- 后台日志、部署参数、运行配置进一步规范。
- 在知识库接入和工具调用方面做了稳态优化，降低异常概率。

## 二、这次升级后，你会直接感受到的改进

- 提问后，能力匹配更准确，结果结构更清楚。
- 处理“政策解读、报告摘要、材料对比”类任务更顺手。
- 上传文件后，可直接围绕文件内容继续追问，不需要反复复制文本。

## 三、你需要做什么

- 正常使用即可，一般不需要额外配置。
- 如果你在处理复杂任务，建议优先使用“思考模式”。
- 如果你只需快速确认数据或口径，建议使用“快速模式”。

## 四、建议提问方式（效果更好）

- 先写清时间范围：例如“2024年”“2025年1-3月”。
- 再写清业务口径：例如“规上工业”“高技术制造业”。
- 明确任务目标：查数据、做分析、画图、导出材料。

## 五、注意事项

- 智能体会优先依据已接入数据源和知识库回答问题。
- 当缺少可用数据时，系统会明确提示，不会编造结果。
- 涉及正式审批、法规适用、最终行政结论时，请以正式流程和权威文件为准。

## 六、后续计划

下一阶段将持续优化以下方向：

- 进一步提升政策检索和知识库问答的准确度。
- 持续优化报告生成质量和图表展示效果。
- 完善更多业务场景下的提问模板与引导能力。

如在使用中遇到问题或有改进建议，请及时反馈给维护团队。感谢大家对经信智能体建设工作的支持。
