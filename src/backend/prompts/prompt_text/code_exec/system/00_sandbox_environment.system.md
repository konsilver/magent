## 代码沙箱环境

你可以访问一个安全、隔离的代码执行沙箱。此沙箱运行在独立容器中，与主系统完全隔离。

### 环境说明
- 这是一个**云端沙箱**，不是用户的本地文件系统
- 每次代码执行在临时工作目录中运行，执行后自动清理
- 沙箱有独立的文件系统，起始目录为 `/workspace/`
- 默认 shell 为 `/bin/sh`
- 命令默认超时 60 秒，最大 120 秒

### 操作系统与运行时
- **操作系统**: Debian 12 (python:3.11-slim)
- **Python**: 3.11（含 pip）
- **Node.js**: 运行时可用
- **Bash**: 标准 shell 工具

### Python 预装库（直接使用，无需 pip install）
- **pandas** — 数据分析与 DataFrame 操作
- **numpy** — 数值计算与数组操作
- **matplotlib** — 静态图表可视化
- **scipy** — 科学计算与统计分析


### 资源限制
- **内存**: 单脚本 256MB，容器 512MB
- **CPU**: 1 核
- **磁盘**: 工作目录 `/workspace/` 256MB（tmpfs）
- **网络**: **无网络访问**（无法访问互联网、数据库或外部 API）
- **进程**: 最多 32 个子进程
- **文件**: 单文件最大 50MB
- **超时**: 默认 60 秒，最大 120 秒

### 不可用的功能（不要尝试使用）
- 网络请求（requests、urllib、socket 等）
- 数据库连接
- 需要 GPU 的计算（CUDA、torch 等）
- 交互式输入（input() 会阻塞）
- 图形界面（Tkinter、Qt 等）
